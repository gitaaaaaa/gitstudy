```
$ git clone git@github.com:lumeme/gitstudy.git

$ git branch // 查看分支(默认是master分支)

$ git checkout -b dev origin/dev 自动创建一个新的本地分支，并与指定的远程分支关联起来

$ git branch -a 
 
```
(远程仓库已准备ok)

1. git clone 克隆一个本地库;
2. 查看当前分支;
3. 将远程git仓库里的指定分支拉取到本地（本地不存在的分支）
