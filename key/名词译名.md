### 专用名词的译名
```
  - Workspace：工作区
  - Index / Stage：暂存区
  - Repository：仓库区（或本地仓库）
  - Remote：远程仓库
```

```
pull 拉 
push 推送
```

```
merge 合并
remote 远程
origin 起源
fork 分叉

checkout 检出
request 需求
fetch 抓取
feature 新功能

pull request 就是一个"请求"(request)
git pull  由两个命令完成  git fetch(拿来) + git merge
```
